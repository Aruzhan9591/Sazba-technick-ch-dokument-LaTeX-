# STD – Semester Project

## Description

This repository contains a semester project created for the STD course at VSB – Technical University of Ostrava.

The document is written in LaTeX and demonstrates the preparation of a professionally formatted technical document. The project showcases various LaTeX features commonly used in academic and technical writing.

## Features

* Title page creation
* Automatic table of contents
* List of figures
* List of tables
* Bibliography management using BibLaTeX
* Source code listings with syntax highlighting
* Mathematical expressions and formulas
* Figure and table formatting
* Cross-references and citations
* Structured academic document layout

## Technologies Used

* LaTeX
* BibLaTeX
* Polyglossia
* Overleaf
* Python code listings

## Project Structure

* `main.tex` – Main LaTeX source file
* `SampleBib.bib` – Bibliography database
* Additional images and resources

## Learning Outcomes

Through this project, I gained practical experience in:

* Technical documentation
* Academic writing and formatting
* Bibliography management
* LaTeX document preparation
* Professional document design
